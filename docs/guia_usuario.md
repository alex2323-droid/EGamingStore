# Guía de Usuario

Esta es la guía de usuario para la aplicación de Egaming Store. Aquí encontrarás instrucciones sobre cómo utilizar la plataforma para realizar recargas de tus juegos favoritos.

## Realizar una recarga
1. Inicia sesión con tu cuenta de Google.
2. Selecciona el juego que deseas recargar.
3. Elige el paquete que más te convenga.
4. Completa la información solicitada (como tu Player ID).
5. Sube tu comprobante de pago.
6. Espera la confirmación de la orden.
